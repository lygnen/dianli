# 电力现货价格预测系统 Code Wiki

## 1. 项目整体架构 (Project Architecture)

本项目是一个基于多种机器学习和深度学习算法的**日前电价（Day-Ahead Electricity Price）**预测系统。系统利用风电、光伏、电力负荷以及火电负载等特征，预测未来24小时或特定时段的电价。

整个仓库包含四个相互独立的建模和预测模块，针对不同的业务需求（如全天独立预测、时序滚动预测、单点深度分析、自回归填补）采用了不同的算法策略：

1. **基于随机森林的24点独立预测** (`main.py`)
2. **基于LSTM的时序滚动预测** (`lstm.py`)
3. **针对特定时段的单点分析预测** (`单点预测.py`)
4. **结合人工特征工程的自回归线性预测** (`自回归.py`, `自回归预测.py`)

## 2. 依赖关系 (Dependencies)

项目依赖于标准的数据科学与机器学习技术栈：

- **数据处理与分析**: `pandas`, `numpy`, `openpyxl` (处理Excel文件的读写)
- **传统机器学习**: `scikit-learn` (随机森林回归、线性回归、数据标准化、性能评估)
- **深度学习**: `tensorflow` (Keras API构建LSTM网络)
- **数据可视化**: `matplotlib`, `seaborn`

*提示: 若需从零运行项目，可以通过 `pip install numpy pandas scikit-learn matplotlib seaborn tensorflow openpyxl` 一键安装。*

## 3. 核心文件与主要模块职责 (Module Responsibilities)

### 3.1 24点随机森林预测 (`main.py`)
- **主要职责**：为一天中的24个时段分别训练24个独立的 `RandomForestRegressor` 模型。该模块的核心理念是“解耦”，即每个小时的预测仅依赖非电价特征，避免了各个时间点电价特征的相互干扰。
- **关键函数说明**：
  - `preprocess_data(df, target_hour)`: 数据预处理。使用中位数填补缺失值，并**剔除**所有其他时段的电价列，确保特征中不包含电价泄漏。
  - `train_models_for_all_hours(df, hour_columns)`: 核心训练流程。循环训练24个独立的模型，提取并打印各时段最重要的5个特征（Feature Importance）。
  - `predict_new_data_all_hours(...)`: 加载待预测数据，使用训练好的24个模型输出完整的24小时电价。
  - `plot_hourly_performance(...)`: 绘制各个时段的MAE、R²、正确率和过拟合程度的可视化图表。
- **数据流**：读取 `随机森林.xlsx` (训练/预测 Sheet) $\rightarrow$ 输出 `24点电价预测结果_[时间戳].xlsx` 和 `24个时段模型性能分析.png`。

### 3.2 LSTM时序滚动预测 (`lstm.py`)
- **主要职责**：利用长短期记忆网络 (LSTM) 捕获电价的序列依赖性，结合前一天的电价以及当天的风、光、负荷数据进行24小时**滚动预测**（用前 $N$ 天预测第 $N+1$ 天）。
- **关键设计**：根据电价波动特性，将全天划分为 **“非中午” (0-8点, 17-23点)** 和 **“中午” (9-16点)** 两个不同性质的时段，分别建立子模型。
- **关键函数说明**：
  - `add_price_lag_features(price_df, lag=1)`: 构造滞后特征（提取前一天电价作为当前特征）。
  - `create_lstm_model(input_shape, output_steps)`: 使用Keras构建LSTM网络（32个单元，`tanh`激活，带20% Dropout），通过 `Adam` 优化器编译。
  - `extract_period_data(X, y, hour_ranges)`: 根据“中午”和“非中午”时段拆分数据。
- **数据流**：读取 `汇总.xlsx` 的五个Sheet $\rightarrow$ 逐日滚动训练和预测 $\rightarrow$ 输出 `滚动预测结果_简化特征.xlsx` 及对比图。

### 3.3 单点预测深度分析 (`单点预测.py`)
- **主要职责**：针对特定的高价值或高波动时段（如上午9点），进行深入的随机森林回归分析。它比 `main.py` 提供了更详尽的诊断和分析工具。
- **关键特征**：不仅预测数值，还通过分析随机森林中各棵决策树 (`rf_model.estimators_`) 的方差，计算出**预测标准差**和**预测置信度**。
- **关键函数说明**：
  - `calculate_accuracy(...)`: 计算特定偏差阈值（默认5元）内的预测正确率。
  - `save_training_predictions(...)`: 汇总特征、实际值、预测值、偏差、置信度等指标并存入详细表。
- **数据流**：读取 `9点随机森林.xlsx` $\rightarrow$ 输出 `训练集预测结果详细表.xlsx`，`随机森林_预测结果.xlsx` 并绘制多维度性能图（散点图、误差分布图、分箱准确率等）。

### 3.4 自回归线性预测体系 (`自回归.py` & `自回归预测.py`)
- **主要职责**：
  - **`自回归.py`**: 基于历史经验和相关性分析，构建定制化特征工程的24小时线性回归（`LinearRegression`）模型，主要用于在测试集上评估该策略的MAE等指标。
  - **`自回归预测.py`**: 应用上述逻辑，专门用于**识别并填补**历史数据中的电价缺失值。
- **算法设计与特征工程**：
  - **自回归特征**：引入**前一小时的预测电价**和**前一天相同时段的电价**作为关键特征。1点电价预测甚至引入了前一天24点的电价及23-24点变化率。
  - **特殊点定制 (special_features)**：通过硬编码配置字典，对预测难度大的特定时段（如6, 7, 8, 18, 19, 24点）采用特殊的时间偏移特征。例如6点模型强制使用了1点/2点的光伏数据和9点/8点的风电数据。
- **数据流**：读取 `汇总.xlsx` $\rightarrow$ 模型训练与滚动推理 $\rightarrow$ 输出 `滚动测试结果_优化点.xlsx` 或填充完整的 `电价预测结果_完整.xlsx`。

## 4. 数据字典 (Data Dictionary)

项目中涉及的Excel数据源主要有以下几个：

1. **`汇总.xlsx`**：时间序列全集数据，包含5个工作表（Sheet）：`日前电价`、`风电24`、`光伏24`、`负荷24`、`负载24`。主要被 LSTM模型和自回归模型使用。
2. **`随机森林.xlsx`**：按行组织的特征宽表数据，包含 `训练` 和 `预测` 两个Sheet，每一行包含当天的风光负荷特征及24点电价。被 `main.py` 使用。
3. **`9点随机森林.xlsx`**：专门用于某单点时段建模的特征表。被 `单点预测.py` 使用。

## 5. 项目运行方式 (How to Run)

### 5.1 环境初始化
在运行代码之前，请确保 Python 环境中安装了以下库：
```bash
pip install numpy pandas scikit-learn matplotlib seaborn tensorflow openpyxl
```

### 5.2 运行不同模块
所有代码文件均可直接作为主脚本执行，脚本会自动读取本地目录下的 Excel 数据并输出结果：

1. **执行24小时全天独立预测** (基于随机森林)
   ```bash
   python main.py
   ```
   *将生成带有时间戳的预测结果表以及性能分析折线图。*

2. **执行时序滚动预测** (基于LSTM)
   ```bash
   python lstm.py
   ```
   *控制台将打印每天的滚动预测进度，生成 `滚动预测结果_简化特征.xlsx`。*

3. **执行单点深度诊断分析** (如9点分析)
   ```bash
   python 单点预测.py
   ```
   *除了预测结果，还会生成预测置信度表并展示6个不同的诊断图表。*

4. **执行自回归模型测试评估**
   ```bash
   python 自回归.py
   ```
   *按9:1划分训练/测试集，评估自回归特征和特殊点优化的效果。*

5. **执行自回归缺失值填补** (生产环境应用)
   ```bash
   python 自回归预测.py
   ```
   *自动识别 `汇总.xlsx` 中的空缺电价天数，利用训练好的自回归模型进行填补，输出 `电价预测结果_完整.xlsx`。*
